vue基础

发表于 2021-01-22 更新于 2021-03-27

基本了解

1.1，初识hello world

  1. script脚本形式引入vue脚本
<script src="https://cdn.bootcdn.net/ajax/libs/vue/3.0.2/vue.global.prod.js"></script>
  1. 对DOM里的指定标签指定渲染
Vue.createApp({
  data() {
    return {
    	content: 1,
  };
  },
  mounted() {
    setInterval(() => {
      this.content += 1;
      //写法二：this.$data.content += 1;
    }, 1000);
  },
  template: "<div>{{content}}</div>",
}).mount("#root");
  • mount指定行为绑定那个DOM
  • data函数表示数据
  • mounted函数：类似生命周期函数componentDidMount，在标签渲染后自动执行
  • template: （模板）渲染到DOM内的内容
